# BuildX India 2026

BuildX India is a premier national-level online hackathon designed to empower the next generation of Indian developers. Organized by **UDTech India**, this 24-hour innovation marathon brings together brilliant minds to solve real-world challenges using cutting-edge technology.

We believe that innovation has no boundaries. Whether you're a student developer from a Tier-1 city or a self-taught coder from a remote village, BuildX provides the platform to showcase your skills, learn from industry experts, and build solutions that matter.

## 🚀 The Mission

Our goal is simple: **To identify and nurture technical talent across India.** BuildX India 2026 isn't just about writing code; it's about building products that solve problems, fostering a culture of collaboration, and connecting talent with opportunity.

## 🛠️ Tech Stack & Architecture

The BuildX India platform is built with performance, scalability, and modern aesthetics in mind:

- **Frontend:** React with TypeScript for type-safe, robust code.
- **Build Tool:** Vite for lightning-fast development and optimized production builds.
- **Styling:** Tailwind CSS for a modern, utility-first design system.
- **Animations:** Framer Motion (motion/react) for smooth, high-performance UI interactions.
- **Icons:** Lucide React for a clean, consistent iconography.

## 🚦 Getting Started

To run the project locally for development or contributions:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/udtechindia/buildx-india-2026.git
    cd buildx-india-2026
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```
